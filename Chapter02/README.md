# Chapter 2 Code Examples

This folder contains the runnable code examples for Chapter 2.

## Examples

* **shoppingCart.js:** Demonstrates the "Before" and "After" refactoring of the shopping cart function.
    * To run: `node shoppingCart.js`
* **caching.js:** Demonstrates the "Before" and "After" of adding a caching layer.
    * To run: `node caching.js`
* **decisionEngine.py:** The runnable Python script for the weighted decision model.
    * To run: `python decisionEngine.py`

* **reportGenerator.js**: A self-contained Node.js project. See the `README.md` inside that folder for instructions.
