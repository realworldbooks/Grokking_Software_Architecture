# Chapter 2 Code Examples

This folder contains the runnable code examples for Chapter 2.

## Examples

* **maintainability/shoppingCart.js:** Demonstrates the "Before" and "After" refactoring of the shopping cart function.
    * To run: `node shoppingCart.js`
* **performance/caching.js:** Demonstrates the "Before" and "After" of adding a caching layer.
    * To run: `node caching.js`
* **weighted_decision_model/decisionEngine.py:** The runnable Python script for the weighted decision model.
    * To run: `python decisionEngine.py`

* **testability/reportGenerator.test.js**: A self-contained Node.js project. See the `README.md` inside that folder for instructions.
